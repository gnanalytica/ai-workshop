> 🧠 **Quick glossary.**
> **Token** = roughly 4 characters of English (less in Hindi/Telugu — closer to 2). The unit OpenAI/Anthropic bill you in.
> **Input vs output tokens** — billed separately. Output is 3–5× costlier (the model is *generating*, not just reading).
> **Closed API** = OpenAI / Anthropic / Gemini. You pay per token. They host.
> **Open source** = Llama (Meta), Mistral, Qwen, DeepSeek. Free to use. You host (or pay someone else to).
> **GPU** = the chip that runs models. NVIDIA owns ~90% of this market.
> **Neo Cloud** = newer GPU clouds (RunPod, Lambda, Together, CoreWeave) that undercut AWS by 3–5× on hourly rates.
> **Knowledge Graph (KG / GraphRAG)** = structured facts that replace stuffing the whole document into the prompt. Less input tokens = less ₹.

### The cost equation, plain

```
cost_per_request = (in_tokens × in_price) + (out_tokens × out_price)
monthly_cost     = users × requests/user/day × 30 × cost_per_request
break_even       = (gpu_hourly × 720) / cost_per_request
```

That's it. Three lines. Plug in your numbers.

### The three levers that actually move the bill

1. **Cut input tokens.** Move from "stuff the whole PDF" to a Knowledge Graph or RAG. 60–90% saving.
2. **Use the smaller model where you can.** Haiku / 4o-mini handles ~70% of traffic at 5× lower cost. Route the hard 30% to Sonnet / 4o.
3. **Cache aggressively.** Anthropic's prompt caching = **90% off cached input tokens**. System prompts are *always* cacheable.

### Indian-context numbers (May 2026, USD → INR ≈ 85)

- **Claude Sonnet 4.6**: ~₹255 / 1M input · ~₹1,275 / 1M output ($3 / $15).
- **Claude Haiku 4.5**: ~₹85 / 1M input · ~₹425 / 1M output ($1 / $5). Hits 70% of your traffic at 5× lower cost.
- **GPT-4o-mini**: ~₹12 / 1M input · ~₹50 / 1M output. Cheap router target.
- **RunPod A100 80GB**: ~₹60/hr community / ~₹100/hr on-demand.
- **Lambda Labs H100**: ~$2.49/hr ≈ ~₹211/hr.
- **Llama-3.3-70B on Together AI**: ~₹75 / 1M combined tokens (single-tier billing).
- **Anthropic prompt caching**: cached input read at 10% of base price (= 90% off). System prompts → **always cache**.
