## Part 2 — Adding the AI Chatbot

### Concept: How the chatbot works

```
User types a question in chat widget
        ↓
Next.js sends it to Google Gemini API
        ↓
Gemini reads your system prompt (knows it's AirDosa assistant)
        ↓
Gemini replies with an answer
        ↓
Answer appears in chat widget on your site
```

### Step 1 — Get your Gemini API key
- Go to aistudio.google.com
- Sign in with Google → go to **Get API Key**
- Create new key → copy it → save it safely
- It looks like: `AIzaSy...`

### Step 2 — Add to .env.local
```
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
GEMINI_API_KEY=AIzaSy-your-key-here
```

### Step 3 — Antigravity prompt for chatbot

```
Add a floating AI chatbot widget to this Next.js app using the Google Gemini API.
The chatbot should know it is the AirDosa assistant — it delivers hot dosas by drone.
It should answer questions about the menu (Classic Masala, Cheese Burst, Gunpowder Ghee Roast),
delivery time (15-20 minutes), pricing (Standard, Express, Dosa Enthusiast plans),
and how drone delivery works.
Show a chat bubble icon fixed at bottom right of the page.
Clicking it opens a chat window.
Create an API route at app/api/chat/route.js that calls Gemini using the
GEMINI_API_KEY environment variable.
```

### Step 4 — Deploy chatbot to Vercel

**Test on live site:**
- Open your live Vercel URL
- Click the chat bubble bottom right
- Ask: "What dosas do you have?" → Claude should reply ✅
