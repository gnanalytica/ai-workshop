1. AirDosa HTML converted to a Next.js 14 App Router project.
2. A live Supabase orders table imported from Excel CSV.

4. Everything deployed live on Vercel with auto-deploy on every GitHub push.

### Antigravity prompts
 
**Step 1 — Convert to Next.js:**
```
Convert this single HTML file into a Next.js 14 App Router project using pnpm.
Keep all existing CSS, animations, fonts, and JS behavior exactly the same.
Create: app/page.js, app/globals.css, app/layout.js.
Replace any Lucide CDN script tag with the lucide-react npm package.
```
 
**Step 3a — Add Supabase:**
```
Add Supabase to this Next.js project. Create lib/supabase.js using
environment variables NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY.
```
 
**Step 3b — Wire Order Now button:**
```
When the Order Now button is clicked, open a modal asking for the user's name,
item and delivery address. On submit, insert a row into the Supabase orders table.
Show a success message after.
```
 
### .env.local (do not push to GitHub)
```
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```
