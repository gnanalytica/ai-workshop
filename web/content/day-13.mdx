> 🧠 **Quick glossary.** **Workflow** = a chain of steps. **Trigger** = what starts it (cron, webhook, manual). **Node** = one step. **Webhook** = a URL that, when hit, runs the workflow. **Scrape** = pull HTML; **extract** = turn it into structured fields. **Idempotent** = safe to run twice.

## 🧪 Lab: Wire one real automation

1. **Pick the task.** Example: "Every morning, give me a list of new internships matching X on Y site."
2. **Trigger.** Add a *Schedule* node, set to daily 9:00 IST.
3. **Scrape.** Add an *HTTP Request* node calling Firecrawl `/scrape` with your URL → markdown out.
4. **Extract.** Add an *OpenAI / Anthropic* node. Prompt: *"From this page markdown, return JSON: [&#123;title, location, link&#125;]"*. Use structured output.
5. **Filter.** *IF* node: only items containing your keyword (e.g., "Bengaluru").
6. **Notify.** *Gmail* / *Slack* / *Telegram* node → send the digest to you.
7. **Run manually once.** Then activate the schedule.

**Artifact.** Screenshot of the activated workflow + the first message it sent you. Submit on dashboard.

> ⚠️ **Don't get banned.** Respect robots.txt. Don't poll faster than once per minute. Use Firecrawl's caching.
