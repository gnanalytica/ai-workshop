# Archive — pre-Next.js plans and specs

These plans and specs describe the pre-Next.js static-HTML era of this project (`faculty.html`, `admin-faculty.html`, `assets/faculty-tabs.js`, hash-routed tabs, vanilla ES modules under `/assets/`). They are preserved for historical context only and no longer reflect the codebase. The current implementation lives under `web/` as a Next.js 15 App Router application against Supabase. For the up-to-date architecture and conventions, see `CLAUDE.md` at the repo root.
